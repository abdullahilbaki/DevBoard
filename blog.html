Question-1: What are the different ways to select an element in the DOM?

Elements on a webpage can be selected in different ways. The most common method is by ID (document.getElementById("myId")), which is like calling someone by their unique name. Another way is by Class (document.getElementsByClassName("myClass")), which selects a group of similar elements. Selecting by Tag (document.getElementsByTagName("p")) is useful when targeting all elements of the same type, like all buttons or paragraphs.

A more flexible way is using querySelector (document.querySelector(".myClass")), which allows selecting elements the same way CSS does. For multiple elements, querySelectorAll (document.querySelectorAll(".myClass")) can be used.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

These three are used to get or change the text inside an element, but they work differently:

1. innerHTML - Gets or sets the content with HTML tags.
2. innerText - Gets only visible text, ignoring hidden elements.
3. textContent - Gets all text, even if it's hidden.

Question-3: What is event delegation in the DOM?

Instead of adding an event to every small element, the event can be placed on a parent element. This way, when anything inside it is clicked, the parent recognizes the action. This method is useful when handling many similar elements, like a list of buttons or items. It makes the code cleaner and more efficient.

Question-4: What is event bubbling in the DOM?

When an event happens on an element inside another element, it first triggers on the smaller one and then moves up to the bigger ones. This process is called event bubbling.

For example, clicking a button inside a div first affects the button, then the div, then the entire page. This behavior can be useful in some cases but may also need to be controlled to prevent unwanted effects.

Question-5: How do you create, add, and remove elements using JavaScript?

If we want to create a new element (like a <p>, <div>, etc.), we use document.createElement("tagname").

For putting it inside an existing element, we can use .appendChild() or .append(), like document.body.appendChild(document.createElement("p")).

To delete an element, we use .remove(), like document.getElementById("myDiv").remove().
